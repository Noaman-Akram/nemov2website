// MARQUEE SIZING LAB

Three ways to make mixed logo sets feel balanced.

Same six brands, three sizing systems. The goal is not mathematical equality. The goal is that every logo feels visible, intentional, and calm when it moves through the strip.

Tiered Buckets

Simple to maintain. Good when the list changes and you want a clear system instead of per-logo tuning.

Visual Weight Multipliers

Best for curated client sets. Each logo gets its own image-level calibration so the visible mark, not the file box, feels balanced.

Normalized Frames

Most forgiving when source files are messy or padded inconsistently. The slot creates rhythm for you.

Retool Stack Motion

Opposing rows with deeper travel. More kinetic and product-y than the standard single-strip marquee.

// OPTION A

Tiered Buckets

Three semantic sizes: dense logos step down slightly, thin wordmarks step up, everything else sits in the middle.

Fastest to maintain when logos change often.

// OPTION B

Visual Weight Multipliers

This version ignores equal containers and tunes the raw logos directly. Heavy marks come down, weak files get pushed up, and the strip reads more evenly at a glance.

Recommended for the final marquee.

Why it works It corrects what you actually see. Huawei stops dominating, and the smaller government mark plus EIEC get enough physical size to stay readable in motion.
What changes per logo Height and max-width are tuned per image. That gives more control when the source PNG bounds are inconsistent or padded badly.
What I would ship This one, with one more round of visual nudging after you react to the current screenshot.
// OPTION C

Normalized Frames

Each logo sits in a quiet slot. The frame absorbs inconsistent source padding and makes the rhythm more even without having to tune every logo aggressively.

Safest when the incoming files are inconsistent.

// OPTION D

Retool Stack Motion

Pulled from the Retool save in your archive: opposing rows, deeper translate ranges, and a more active product-brand rhythm than a standard logo rail.

Best when you want motion to feel more alive.

Archive check: Realm gave a static wrapped logo list, not a true marquee. Retool gave the usable motion reference: opposing rows and deeper translate ranges.
