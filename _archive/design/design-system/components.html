T2 — 2×2 feature grid

Token-driven

Every value — spacing, colour, type, radius — lives in a token. Redesign at scale with a single variable change.

Responsive by default

Layouts flex gracefully from 320px to 4K. Nothing breaks. Nothing looks like an afterthought.

Accessible

WCAG AA contrast ratios, focus states, and semantic HTML are built in — not bolted on.

Documented

Every component ships with usage notes, variants, and do/don't guidance so your team can move fast.
